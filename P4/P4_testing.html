4. Tester Audience

The navigation structure of the website is quite simple. As shown in the figure below, you can reach any page with one click from any other page on the site. This also includes using a hoverable drop-down menu for the general gallery aka products.

The reason we chose this structure is mainly that we want the user to be able to use the header to reach any page he/she desires at any given point, and that he/she will know where to look in order to find the desired page.
